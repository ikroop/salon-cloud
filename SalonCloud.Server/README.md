Build project
=============

Run web server
=============
```
$ npm install -g nodemon
$ cd <project_path> (ex: workspace/salon-cloud/SalonCloud.Server)
$ npm install 
$ nodemon ./app.js
```
Run Unit Test
=============
```
$ npm install -g mocha
$ mocha ./spec/authentication.spec.js
```
